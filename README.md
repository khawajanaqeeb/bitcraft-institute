## 🛠️ Tech Stack

This project is built with a modern, scalable frontend stack:

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + CSS Variables
- **Animations:** Framer Motion + Custom CSS animations
- **Theming:** Light/Dark Mode (CSS variable–based)
- **Fonts:** Geist Sans & Geist Mono
- **Bundler:** Turbopack (Next.js)

---

## 🎨 UI & Design Highlights

- Glassmorphism UI components
- Smooth page & section animations
- Fully responsive design
- Dark mode with system & manual toggle
- Accessible typography and color contrast
- Professional gradient & modern layouts

---

## 🌗 Dark Mode Support

- Class-based dark mode (`html.dark`)
- CSS variables for full theme control
- Persistent theme preference using `localStorage`
- No hydration mismatch or flashing

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
